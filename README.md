Meu Portfólio Pessoal (Desenvolvimento & Dados)

Este é o meu site de portfólio oficial, desenvolvido originalmente em 2023 para centralizar e apresentar os meus melhores projetos de desenvolvimento de software e análise de dados. É aqui que normalmente armazeno meus projetos.

## 🤝 O que eu fiz nesse projeto
Criei e arquitetei toda a estrutura do site do zero, passando por duas etapas principais:
* **Design e Prototipagem (Figma):** Antes de encostar no código, pensei em toda a ideia, desenhei telas,  o mapa de navegação e a identidade visual direto no Figma para garantir uma interface limpa e intuitiva.
* **Desenvolvimento Web:** Traduzi o layout do Figma para código limpo usando HTML5 semântico para a estrutura, CSS3 para deixar o layout totalmente responsivo (adaptado para celular e computador) e JavaScript puro (Vanilla JS) para os efeitos e interações. Encontrei muitas dificuldades nessa etapa, mas com as pesquisas e estudos necessários, consegui melhorar ainda mais meu projeto.

## 🛠️ O que tem na estrutura do código
Para o projeto não ficar bagunçado, estruturei os arquivos de forma bem organizada:
* `index.html`: Toda a marcação estrutural e semântica do site.
* `script.js`: A lógica e interações dinâmicas da página.
* `css/` ou arquivos de estilo: Toda a identidade visual e responsividade.
* Documentos e mídias essenciais do meu histórico profissional.

## 💡 Evolução contínua
Este projeto nasceu em 2023 como uma base de front-end e, hoje, serve como ponto de partida para conectar minhas soluções mais avançadas de engenharia e análise de dados (como motores de recomendação em Python e bancos de dados orientados a grafos).
